# i2x-code-challenge
code challenge project

## Running the Project Locally
`yarn install`

`yarn dev`

## Deploying the Project

`yarn build`

`heroku create`

`git add .`

`git commit -m "<YOUR_COMMIT_MESSAGE>"`

`git push heroku master`

`heroku open`
